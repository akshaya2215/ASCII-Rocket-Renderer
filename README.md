# 🚀 ASCII Rocket Renderer

A software-rendered 3D ASCII rocket built entirely in C++ without using graphics libraries.

## 🎯 Goal

This project explores the fundamentals of computer graphics by rendering and animating a rocket directly in the terminal using ASCII characters.

## ✨ Planned Features

- [ ] Terminal screen buffer
- [ ] Pixel drawing
- [ ] Line drawing
- [ ] Wireframe rocket
- [ ] 3D rotation
- [ ] Perspective projection
- [ ] Lighting and shading
- [ ] Animated rocket flames
- [ ] Star field
- [ ] Camera controls

## 🛠️ Technologies

- C++17
- Object-Oriented Programming
- Mathematics
- Trigonometry
- Linear Algebra
- Git & GitHub
